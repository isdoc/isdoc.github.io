ISDOC – Czech national standard for e-invoicing

Version 6.0.2 from 2022-03-23

Copyright © 2022 MVČR

Copyright © 2014 ICT UNIE o.s.

Attached schemas and standard formally define exchange format for invoices and other types of documents. The format was originally created by Working Group for Electronic Data Interchange Standards of ICT Unie o.s.. As of now future development and maintenance is done by Ministry of the Interior.

Inquires can be sent to isdoc@mvcr.cz.

Changelog

Changes since version 6.0.1 from 2014-05-26

  • Added a new document representation ISDOC.PDF that allows embedding of ISDOC into PDF/A-3 format.

  • Description of some elements was improved.

  • Schema now enforces uniqueness of line identifier.

  • Clarified details about time stamping.
